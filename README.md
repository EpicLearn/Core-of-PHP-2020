# 介绍

当您对 PHP 及其核心框架程序集有疑问时，本开发指南提供了您需要的答案。  
自 1995 年首次亮相以来，PHP 已经成为一种具有不同寻常的灵活性和广度的语言，但它的持续增长意味着总是有更多的东西需要学习。
本书围绕概念和用例组织，为中级和高级程序员提供了 PHP 知识的简明路线。
